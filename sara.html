Username

nome de usuário | (ˈyo͞ozərˌnām) | Se refere ao nome único que identifica um usuário em um determinado sistema, plataforma ou serviço online.

Vírus

Vírus (ˈvīrəs) É um tipo de software malicioso, ou malware, que se espalha entre computadores e causa danos aos dados e ao software.

Windows

Windows (win.dowz) É um sistema operacional de multitarefas para computadores e dispositivos móveis desenvolvido pela Microsoft.

Bluetooth

Bluetooth (blu.tooth) É um padrão de tecnologia sem fio de curto alcance usado para troca de dados entre dispositivos fixos e móveis em distâncias curtas e construção de redes de área pessoal (PANs).

Network

Rede ˈnetˌwərk Um grupo de computadores interconectados capazes de trocar Informação. Uma rede pode ser tão pequena quanto várias pessoas computadores em uma LAN ou tão grandes quanto a Internet, um mercado mundial rede de computadores.
